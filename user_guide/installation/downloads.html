Servidor Git

Git es un sistema de control de versiones distribuido.

El acceso público a Git está disponible en GitHub. Tenga en cuenta que, si bien hace todo lo posible para mantener esta base de código funcional, no podemos garantizar la funcionalidad del código tomado desde el repositorio.

Comenzando con la versión 2.0.3, la etiqueta "stable" también está disponible vía BitBucket simplemente seleccionando la versión en la lista desplegable "Tags".
